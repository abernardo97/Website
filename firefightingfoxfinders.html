Project Description

Firefighting Fox Finders is a first person game that was developed in Unity in C#.

The forest is on fire! Foxes are scattered throughout the forest in need of your help. Carefully maneuver around the fire to pick up the foxes to save them. Dont worry - you can put out the fires for a short time.

Project Details

  • Created during HeroJam 2017
  • Developed in Visual Studio 2017 in C#
  • Built with Unity 5.6
  • 2D assets made in Photshop
  • 3D assets modeled in Maya

Project Contribution
