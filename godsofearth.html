Gods of Earth Dec 2016 View Project

Project Description

Gods of Earth is a turn-based puzzle strategy table-top board game.

Players compete within their families to manipulate the world so that the human inhabitants develop as the player desires. Each player has a different agenda, so watch out for them causing havoc that can disrupt your own plans.

Project Details

  • Developed for a class project
  • Rulesheet made in Microsoft Word
  • Board pieces built with cardboard
  • Visual assets made in Photoshop

Project Contribution

  • Designed and edited the rulesheet
  • Aided in designing game mechanics
  • Conducted several playtests
