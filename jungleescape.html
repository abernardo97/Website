Jungle Escape Dec 2016

Project Description

Jungle Escape is a 2D puzzle platformer area shooter game.

While you were going to a professional archery competition in South America, your plane crashes in the middle of a wild, unexplored jungle. You must maneuver through the forest to face giant spiders while solving enviroment based puzzles to find your out in a final boss battle.

Project Details

  • Developed for a class project
  • Developed in Visual Studio 2015 in C#
  • Built with the Monogame Framework
  • Visual assets made in Photshop
  • Companion external map editor
