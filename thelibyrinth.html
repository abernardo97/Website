The Libyrinth Oct 2017 View Project

Project Description

The Libyrinth is a 3D stealth maze game that was developed in Unity in C#.

Librarians stalk the halls in search of you! Sneak through the library and don’t get caught. Traps are scattered throughout the maze, where stepping on them will make noise and attract the librarian’s attention.

Project Details

  • Created during IGM Imagine Cup 2017
  • Developed in Visual Studio 2017 in C#
  • Built with Unity 5.6
  • Visual assets made in Photshop
  • Librarian was modeled in Maya
  • Awarded 1st Runner Up

Project Contribution

  • Lead the team as a project manager to maintain consistent communication to deliver a working prototype for each milestone
  • Created all the environmental 2D art assets in Photoshop and then imported it into Unity
