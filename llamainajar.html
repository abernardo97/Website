Llama in a Jar Interactive eCard Jan 2018

Project Description

The Llama in a Jar Interactive ecard is a interactive scene built with THREE.js.

An interactive scene where the player can admire a llama from afar.

Project Details

  • Created during my Fall American Greetings Internship
  • Developed in Visual Studio Code in JavaScript
  • Built with THREE.js
  • Environment assets made in Adobe Photoshop CC 2015
  • 3D Models made in Blender

Project Contribution
