Stella Chaser Feb 2017 View Project

Project Description

Stella Chaser is a surreal 2D runner with a jumping mechanic.

You have been dropped in a surreal 2D infinite runner as Stella, where you "defeat" dark creatures in the night, in a time of darkness.

Project Details

  • Created during Brick Hack 3
  • Developed using Phaser.io in JavaScript
  • Visual assets made in Paint Tool SAI
  • Spritesheets made in CSS Sprites Generator
  • Awarded Best Game Graphics

Project Contribution

  • Lead the team as a project manager to maintain consistent communication to deliver a working prototype
  • Programmed the enemy sprite animations
